# JobOffer-Admin(招聘网站后台管理)

## 简单介绍

&emsp;&emsp;本项目旨在练习React+React-Router-Redux与个人的完全JS编程能力，
UI框架选取主流的Ant-design。项目背景为一个招聘多端网站，后期会发布前台页面，本项目只做后台。

## 技术栈

- React（V 16.9.0）
- React-Router（V 4.x）
- Redux 

## 项目配置

### 克隆项目

```bash
$ git clone https://github.com/bigbigDreamer/JobOffer-Admin
```

### 安装依赖

```bash
$ yarn add 

# or

$ npm i 
```

### 启动项目

```bash
$ npm start

# or

$ yarn start
```

### 打包发布

```bash
$ npm  build

# or

$ yarn build
```
